Paragrahs and Formatting in HTML


Throughout the contents of this sample web page we will go over the basic methods of formatting and declaring custom html paragraphs. You may already have noticed the interestingly placed horizontal line right below our title header declaration. This was done with a simple formatting specifier identified by the tag <hr>. Furthermore, our paragraph is displayed one line lower than it normally would. This white space can be inserted by using the <br> tag.

We can also insert raw pre-formatted text into our web page by using then <pre> tag. Naturally, pre stands for preformatted. We can see an example of the usage of this tag below.


    古池や蛙飛び込む水の音
    
    ふるいけやかわずとびこむみずのおと

    furu ike ya kawazu tobikomu mizu no oto

This is a famous japanese haiku poem that roughly translates to the following: 


    An old silent pond...

    A frog jumps into the pond—

    Splash! Silence again.


There are many more formatting and paragraph tags we may use, among which the following are of note:

  • <strong> - used to bold text
  • <em> - used to italicize text
  • <u> - used to underline text
  • <mark> - used to highlight text
  • <sub> - used to create subscript text
  • <sup> - used to create superscript text

They won't be explained here in order to keep the stylistic tone of this page. Furthermore, these are only a few of the many formatting and paragraph tags we may use in our html documents. For a more exhaustive list, it is recommended to visit the following link: HTML Formatting Tags
