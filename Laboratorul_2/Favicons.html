HTML Lists

We will start with lists because they are a pretty non-problematic and fundamental topic to cover. Additionally, we've already made use of them in previous explanatory pages. As we may guess, there exist two main types of lists in HTML, namely, ordered and unordered lists. The only difference between these two is how the listed elements are indexed. Ordered lists count every listed item. On the other hand, unordered list index elements with bullet points. Unordered and ordered lists are declared as <ul> and <ol> respectively. List items are declared by using the <li> tag.

<!-- This is an unordered list -->
<ul>
<li>First item</li> 
<li>Second item</li>
<li>Third item</li>
</ul>

<!-- This is an ordered list -->
<ol>
<li>First item</li>
<li>Second item</li>
<li>Third item</li>
</ol>

There is also a special type of list, called a definition list, which is used to define terms and their corresponding definitions. Definition lists are declared using the <dl> tag, while terms and definitions are declared using the <dt> and <dd> tags respectively. An example of a definition list is shown below.

<!-- This is a definition list -->
<dl>
<dt>HTML</dt>
<dd>Hypertext Markup Language</dd>
<dt>CSS</dt>
<dd>Cascading Style Sheets</dd>
</dl>

A brief look into CSS

We will now take a look into CSS and how we may use it to improve the aspect of our web pages. As you may or may not be aware, CSS stands for Cascading Style Sheets and it can be quite a mouthful. It is titled as cascading because the style we apply to a parent element will, in turn, be applied to all of associated children. A simple example can be seen in this very document, where a basic CSS style was used to center the text elements of the webpage. We can perform much more interesting and flavorful changes to the style of our page that relate, but are not limited to text alignment, text color(both background and foreground), element borders and element dimensions.

Using CSS

There are three main ways to use CSS in an HTML document, namely, the following.

  • Inline CSS - used to apply style to a single element by using the style attribute within that element's tag.

  • Internal CSS - used to apply style to multiple elements within the same HTML document by using the <style> tag within the <head> section of the document.

  • External CSS - used to apply style to multiple HTML documents by linking an external CSS file using the <link> tag within the <head> section of the document.

There are many CSS style options we can use to alter the aspect of our webpages. A collection of the most frequently encountered options is listed below.

  • color - used to set the foreground color of text.

  • background-color - used to set the background color of an element.

  • text-align - used to set the horizontal alignment of text within an element.

  • font-size - used to set the size of the font.

  • font-family - used to set the font type.

  • border - used to set the border properties of an element.

  • width - used to set the width of an element.

  • height - used to set the height of an element.

    For internal and external CSS, we will supply both the additional code and what we have to do in order to apply its style to our tags. Regarding internal CSS, we are required to define the CSS rules within the <style> tag in the <head> section of the document. For external CSS, we must create a separate .css file and link it to our HTML document using the <link> tag in the <head> section. Below we can find an example of how internal CSS is defined. External definitions are very similar, the only difference being that external CSS is defined in a separate file and linked, as has been specified.

    <style>

.title-center {
    display: block;
    text-align: center;
}

.para-center {
    display: block;
    margin-left: auto;
    margin-right: auto;
    width: 50%;
}

pre.example-box {
    display: block;
    text-align: center;
    margin-left: auto;
    margin-right: auto;
    width: fit-content;
    border: 1px solid black;
}

p {
    font-family: Arial, sans-serif;
    font-size: 14px;
}

</style>

    As we can see in the above example, the CSS style is defined by a name and several properties. The name is used to reference the style in the HTML document, while the properties define how the element should be displayed. There exist two main methods of naming our CSS styles. One of them is the use of class names, which are defined by a period (.) followed by the name of the class. The other method is the use of ID names, which are defined by a hash (#) followed by the name of the ID. Naming CSS styles using a class attribute allows us to apply the same style to multiple elements, while naming them using an ID attribute allows us to apply a unique style to a single element.

    Linking an external CSS document is achieved with the <link> tag in the <head> section of the HTML, likeso.

    <link rel="stylesheet" type="text/css" href="styles.css">
